## Kubernetes Java API示例代码

此模块包含用于演示如何使用Kubernetes客户端Java API的示例代码。

在运行这些示例之前，请确保你的环境已正确配置为访问一个工作的Kubernetes集群。

检查一切是否按预期工作的一种简单方法是执行任何`kubectl get`命令：

```shell
$ kubectl get nodes
```

如果你得到有效的响应，那么你就可以开始了。
