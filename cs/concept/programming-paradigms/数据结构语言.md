## 1. 概述

计算机科学背景下的数据结构和算法是必不可少的，因为这些概念在编程中起着关键作用。大多数软件程序将使用这些概念来操作数据，因此，研究这些方法对于正确理解代码至关重要。

要了解和实施这些想法，我们可以在许多不同的编程语言之间进行选择，每种语言都能够执行必要的任务。为了选择最适合我们需要的工具，本教程将讨论数据结构背后的主要思想，以及它们在当今最流行的编程语言中的改编。

## 2.概述

数据结构是一种以数学方式组织计算机内存空间的方式，可以有效地访问其中的值。因此，数据结构和算法经常一起讨论，因为结构的创建和删除是通过算法仔细引导的。大多数情况下，我们将实现方法来完成数据结构的最基本任务。这些任务可以分为以下几类：

-   插入
-   删除
-   搜索元素

为了完成这些任务，我们使用指针和动态内存分配来管理内存。在面向对象的编程中，内存的管理通常是通过我们的构造函数和析构函数在类级别完成的。这种在我们不需要时释放已用空间的一般想法称为垃圾收集。理想情况下，我们尝试形成高效的算法，只改变完成任务所需的内容。

最后，可以通过时间和空间复杂度分析来计算这些操作的功效。

## 3. 简单链表

需要注意的是，不同的编程语言各有优缺点。我们将通过使用最常见的编程语言实现一个简单的 3 项简单链表来演示这一点，并详细说明它们之间的区别。这个链表是由数据节点组成的基本数据结构。每个节点将保存一些信息和指向下一个节点的指针。

我们将只介绍结构的构造，而忽略完整实现所需的许多其他功能，例如插入、删除、检查列表大小等。

### 3.1. C

我们将从要求明确指定所有变量和对象类型的强类型语言开始。它通常用于嵌入式系统、数据库和驱动程序开发。

首先，我们可以构建我们的主要功能。因为 C 不是面向对象的，所以我们可以使用结构而不是定义类。下面是我们的Node和LinkedList结构的示例：

```c
typedef struct node_t {
    char  data;
    struct node_t  next;
} node;
typedef struct SLinkedList {
    node head;
} linkedList;
```

然后我们可以将节点链接在一起：

```r
node1->next = node2;
```

使用这些指令，我们可以创建一个LinkedList结构，并将其头部指向我们的第一个节点。

C 要求我们明确编码，这使得它成为想要学习如何从头开始实现事物的人的完美语言。但是，它不是面向对象的，它需要我们手动释放我们稍后在代码中用完的内存；这里没有垃圾收集。另请注意，我们必须包括库以处理字符串等。

### 3.2. C++

这种语言被广泛用作许多操作系统的一部分，以及用于制作高效的视频游戏和机器学习工具。它可以看作是C的面向对象版本。这意味着C++将支持抽象、多态、封装和继承。

因此，我们可以用类替换结构。这里的好处是我们可以将分配和取消分配内存空间的函数放在我们的构造函数和析构函数中。这些是生成和销毁我们的对象的函数，它们是自动调用的。

此外，我们可以定义 public 和 private 说明符，因为 C++ 支持封装。这为我们增加了对变量的访问控制。我们的节点类看起来像：

```cpp
class Node {
public:
    char  data;
    Node next;

    Node() {
        data = NULL;
        next = NULL;
    }
  
};
```

和 LinkedList：

```cpp
class SLinkedList {
public:
    SLinkedList(Node firstnode) {
        head = firstnode;
    }
private:
    Node head;
};
```

请注意，头 指针只能通过SLinkedList类的构造函数访问，因为它位于private下。 相比之下，因为我们的实现没有封装Node的next 变量，所以可以通过为它分配一些其他值来破坏链表。

我们可以看到C++的语法与C比较相似，但也有一些例外。C++ 仍然是强类型的，但它比 C 更简洁。

### 3.3. 爪哇

Java是一种基于C++的高级语言。它和C++用在所有相同的领域，我们也可以看出语法非常相似。请注意，这种语言从一开始就包含了很多功能，不需要包含任何外部库。

在这种语言中，我们也不必包含任何库来处理字符串。我们可以通过定义一个类来开始我们的代码，就像在 C++ 中一样。这一次，我们可以将Node类嵌套在LinkedList中：

```powershell
class SLinkedList {
    class Node {
        string data;
        Node next;
        Node(string d) {
            data = d; 
        }
    }
    SLinkedList (Node start) {
    head = start;
    }
}

```

我们可以直接在类中实现构造函数。如果我们为我们的节点创建类并将文本包含在它们的构造函数中，我们可以使用new关键字来定义它们。那么我们在实现链表的时候就可以再次使用这个：

```java
SLinkedList mylist = new SLinkedList(new Node ("First"));
```

最重要的是，所讨论的基于 C 的语言(C、C++、Java)是软件跨平台实现的理想选择，因为大多数机器已经在操作系统中配备了所需的运行时。

### 3.4. Python

Python 是另一种高级语言，广泛用于从机器学习到 Web 应用程序的各个领域。没有必要指定我们为此使用指针，为那个使用字符串，因为该语言是松散类型的。

就像 Java 一样，我们不需要导入任何东西来处理字符串。

首先，我们可以定义一个类：

```python
class Node:
    def __init__(self, dataval = None):
        self.dataval = dataval
        self.nextval = None
```

值得注意的是，这里的缩进非常重要并且定义了函数的顺序。

我们也可以直接在构造函数中定义默认值。然后我们可以 通过调用构造函数来创建Nodes和SLinkedList ：

```python
Node1 = Node("First")
List1 = SLinkedList(Node1)
```

最后，我们可以通过将下一个节点分配给对象来链接节点：

```
Node1.nextval = Node2
```

这使得简单、简洁的语法易于理解。重要的是要注意每条指令后都不需要分号，并且代码上的缩进很重要，因为它们定义了函数和循环的结构。我们也不需要使用任何命令来管理内存空间，因为构造函数和析构函数会分配和释放内存。Python 为我们完成了大部分垃圾收集工作。

这意味着 python 是一种在简化我们的任务方面做了很多工作的语言，它有优点也有缺点。

例如，Python 中的数组会被自动视为链表。因此，我们可以将任何不同的数据类型添加到同一个数组中；它只会保存指向不同内存空间的指针。C 中不是这种情况，因为数组只能在连续的内存空间中保存相同类型的元素。这使得访问 C 中的内存更快，但代价是不能插入不同的数据类型：

![c-vs-python数组](https://www.baeldung.com/wp-content/uploads/sites/4/2021/11/LinkedList-Array-1024x483-1.png)

这意味着如果我们不想定义任何类，或者从头开始实现这个链表，那么完全等价的内容可以总结为一行：

```python
Positions = ["First","Second","Third"]
```

此外，我们可以使用各种不同的预定义方法来获取数组的长度、追加、计数、删除或插入。

### 3.5. Javascript

Javascript 是一种主要用于 Web 开发的编程语言。与 Python 相比，这种语言对细节的关注更为重要。

我们可以从声明我们的类开始：

```javascript
class Node {
    constructor(text) {
        this.position = text;
        this.next = null;
    }
}
class LinkedList {
    constructor(head = null) {
        this.head = head
    }
}
```

我们可以在上面的代码片段中看到，我们必须使用括号来定义函数，并使用分号来结束对机器的每条指令。与其他语言相反，构造函数的定义相当明确。我们也不必在构造函数之前定义position或next 。

就像 Java 一样，创建节点的语法类似于 C++：

```java
let node1 = new Node("First");
```

最后，我们还可以看到我们正在使用new关键字管理内存空间，就像在 C++ 中一样，而在 Python 中则不是这样。

## 4.总结

如前所述，所有这些语言都能够执行基本的数据结构任务。为了选择最好的语言，讨论我们计划编程什么以及如何编程是很有帮助的。现在我们对每种语言的外观有了更广泛的了解，我们可以更准确地做出决定。

如果我们关心的是：

-   对每个基本过程的非常精确的、底层的理解，或者我们想使用嵌入式系统：C
-   机器学习：C++ 或 Python
-   网络：Javascript 或 Python
-   游戏开发：C++、Java 或 Python
-   一些容易编写的东西，几乎没有陷阱和大量的垃圾收集：Python
