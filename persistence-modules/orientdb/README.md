## OrientDB Java API简介

这是一个简单的maven项目，展示了如何使用OrientDB的Java API。

## 要求

- Maven
- Java 7或更高版本
- OrientDB数据库

## 构建

要构建和启动服务器，只需键入

```bash
$ mvn clean install
```

## 运行测试

在启动单元测试之前：

- 安装东方数据库
- 创建BaeldungDB，BaeldungDBTwo和BaeldungDBThree数据库
- 取消注释测试文件上的注释
