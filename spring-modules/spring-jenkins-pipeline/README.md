## 使用Spring Boot的基本CRUD API

这是使用Spring Boot构建的一些CRUD操作的简单API代码。

## 需求

- Maven
- JDK 8
- MongoDB

## 运行

要构建和启动服务器，只需运行以下命令：

```bash
$ mvn clean install
$ mvn spring-boot:run -Dserver.port=8989
```

现在，对于默认配置，它将在以下位置可用：[http://localhost:8080](http://localhost:8080)
