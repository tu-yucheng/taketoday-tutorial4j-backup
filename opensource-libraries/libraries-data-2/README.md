## 构建项目

你可以使用以下方法从命令行生成项目：`mvn clean install`
，或在IDE中生成项目。如果你有问题derive4j导入在IDE中，你必须将文件夹：`target/generated-source/annotation` 添加到项目构建中IDE中的路径。
